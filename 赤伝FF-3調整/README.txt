--------
環境設定
--------
①下のURLからPythonをダウンロード。バージョンは3.14.0がおすすめ。
https://www.python.org/downloads/

②ダウンロードしたファイルでPythonをインストール。

③インストールが終わったら、setup.batをダブルクリックでその他の必要ファイルをインストール。


--------
操作手順
--------
①NetSuiteから対象の請求書の内部IDを取得。
　エクセルでダウンロードすると次の手順②が簡単です。

②取得した内部IDをakaden.xlsxに入力し保存。

③akaden.pyを起動。

④NetSuiteのログイン画面が出るのでログイン。

⑤ログイン出来たらpyの画面に戻りENTERキーを押すと自動処理開始。

⑥処理が終わったらメッセージが出るので、ENTERキーを押して終了。
